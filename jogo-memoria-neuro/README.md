# Memória em Movimento

Jogo de memória estático e responsivo para uma clínica de reabilitação neurológica.

- Não usa imagens, bibliotecas nem fontes externas.
- As cartas usam símbolos nativos do sistema: cérebro, osso, mão, pé e outros.
- A identidade visual atual é apenas uma base provisória para receber o design system da clínica.
- O jogo é educativo e recreativo; não mede capacidade cognitiva nem substitui avaliação profissional.

Para testar, abra `index.html` em um navegador. Para publicar, a pasta pode ser hospedada diretamente em GitHub Pages, Netlify ou Vercel.
